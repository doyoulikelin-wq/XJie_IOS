# Xjie 功能总览

> 版本：v1.6.0 | 更新日期：2026-04-22  
> 定位：面向代谢慢性病患者的智能健康管理 iOS App

---

## 一、身份认证

| 功能 | 说明 |
|------|------|
| 手机号注册/登录 | 手机号 + 密码认证 |
| JWT 自动刷新 | Access Token 30 分钟 + Refresh Token 7 天，401 自动刷新重试 |
| Keychain 安全存储 | Token 存储于系统 Keychain，非 UserDefaults |
| AI 隐私授权 | 首次使用 AI 功能前弹出隐私授权同意，403 时自动触发授权流程 |

---

## 二、首页仪表盘

| 功能 | 说明 |
|------|------|
| KPI 指标卡片 | 今日血糖均值、TIR 达标率、膳食热量、AI 交互次数等关键指标 |
| 干预等级滑块 | L1 温和 / L2 标准 / L3 积极，实时调节 AI 主动推送频率 |
| 离线缓存 | 网络不可用时自动展示上次缓存数据，标记 `isOfflineData` |
| 断网 Banner | 全局顶部提示"网络不可用" |

---

## 三、血糖追踪

| 功能 | 说明 |
|------|------|
| CGM 数据导入 | 支持 Clarity CSV 等格式批量上传 |
| 实时血糖曲线 | SwiftUI Charts 绘制 24 小时 / 7 天趋势图 |
| 血糖统计分析 | TIR 百分比、平均血糖、最高/最低值、变异系数(CV) |
| 日期范围查询 | 按时间窗口筛选血糖数据点 |
| 交互 Tooltip | 点击/拖拽图表查看精确日期 + 血糖值，异常值红色标记 |

---

## 四、膳食记录

| 功能 | 说明 |
|------|------|
| 拍照识别 | Kimi K2.5 多模态视觉识别食物名称 + 预估热量 |
| 手动录入 | 手动添加膳食记录 |
| 分页列表 | 20 条/页，支持加载更多 |
| 膳食-血糖关联 | AI 对话中自动关联餐前/餐后血糖数据分析 |

---

## 五、健康数据管理

| 功能 | 说明 |
|------|------|
| 体检报告上传 | 支持图片/CSV 批量上传，LLM 自动提取结构化数据 |
| 病历记录上传 | 同上，分类为 exam_report / medical_record |
| AI 文档摘要 | 上传时自动生成 ≤10 字 brief + 详细 summary |
| 懒加载生成 | 历史文档首次访问时自动补充生成 AI 摘要 |
| 异常指标标记 | 异常值自动标红（列表 Badge + 详情 Banner） |
| 摘要/原件切换 | 详情页默认展示 AI 摘要，"查看原件" 按钮切换原始 CSV 表格 |
| 文档删除 | 长按 contextMenu 删除 |

---

## 六、AI 对话助手「小捷」

| 功能 | 说明 |
|------|------|
| 温暖医学顾问风格 | 像懂医学的朋友，而非冷冰冰的 AI |
| 健康数据感知 | 自动融合用户血糖、膳食、体检数据进行分析 |
| 结构化输出 | summary(1-2 句) + analysis(详细 Markdown) + followups(3 条追问) |
| 可展开分析气泡 | 简洁摘要 → "查看详细分析" 展开完整分析 |
| 用户画像自动提取 | AI 从对话中识别性别/年龄/身高/体重，写入个人资料 |
| 流式输出 | SSE 事件流实时打字效果 |
| 多会话管理 | 支持新建会话、切换历史会话、分页加载历史消息 |
| 技能匹配 | 根据用户提问关键词自动注入专业 prompt（血糖分析/膳食建议/报告解读等） |
| 功能开关守卫 | `ai_chat` 开关关闭时返回 503，优雅降级 |

---

## 七、智能代理干预

| 功能 | 说明 |
|------|------|
| 每日简报 | 早晨推送当天血糖 + 膳食摘要 + AI 建议 |
| 周评总结 | 每周一次，汇总平均血糖、达标率、改进建议 |
| 餐前模拟 | 预测下顿饭对血糖的影响 |
| 血糖救援 | 高血糖时主动推送降血糖方案 |
| APNs 推送 | Apple HTTP/2 推送通知，JWT 认证 |
| 分级推送频率 | L1 温和 3-4 条/周 → L2 标准 1.5 条/天 → L3 积极 3 条/天 |

---

## 八、健康摘要与指标趋势

| 功能 | 说明 |
|------|------|
| AI 研究报告 | 6 阶段生成：概览→血糖→膳食→多组学→建议→综合 |
| 异步任务追踪 | 后台线程执行，前端轮询进度 + Token 消耗统计 |
| 指标趋势图 | 按指标分组绘制时序折线，异常值红色标注 |
| Tooltip 交互 | 点击查看日期 + 数值，`yyyy年M月d日` 格式 |

---

## 九、多组学数据

| 功能 | 说明 |
|------|------|
| 代谢组上传 | 文件上传 + LLM 解读 + 风险等级评估 |
| **演示模式（v1.6）** | Settings 一键开启，展示完整五幕组学剧本 |
| 代谢健康卡 | 大健康分环 + 4 类系统占比条（心血管/糖代谢/炎症/神经）+ 重点关注 pills |
| 三系统联动卡 | 代谢×血糖×心率三系统健康度并排 + 箭头表达传递 + 一句话洞察 |
| 肠道菌群分布图 | 气泡图按相对丰度可视化，菌属中文名展示，无重叠布局 |
| 基因风险时间轴 | 横向年龄轴 + 风险等级颜色编码 + 点击查看故事 |
| 代谢物故事卡 | 弹出层显示参考范围、状态、关联文献引用 |
| 文献子库 | 70+ 篇组学专题文献本地子库，每周一 03:00 自动增量抓取 |

## 十一、管理后台

### iOS 管理面板（5 个 Tab）

| Tab | 功能 |

| Tab | 功能 |
|-----|------|
| 概览 | 统计卡片 + 4 个数据表格 |

| 技能管理 | 6 个 AI 技能 CRUD（优先级 + 关键词触发 + prompt 模板） |
